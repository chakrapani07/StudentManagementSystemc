# Student Management System

This is a simple Student Management Web Application built with *Spring Boot*, implementing CRUD functionality to efficiently manage student data.  
The project demonstrates clean architecture using *Spring MVC, **Spring Data JPA (Hibernate), **MySQL, and **Thymeleaf* for server-side rendering.

## Technologies Used

### *Backend*
- **Java** – Used to write the application logic.
- **Spring Boot** – Simplifies application setup and provides a production-ready environment.
- **Spring MVC** – Implements the MVC architecture for routing, request handling, and connecting backend with frontend.
- **Spring Data JPA (Hibernate)** – ORM framework used to perform CRUD operations with minimal boilerplate code.

### *Frontend*
- **Thymeleaf** – Server-side template engine for generating dynamic HTML pages.
- **Bootstrap** – Modern, responsive UI components for a clean user interface.

### *Database*
- **MySQL** – Relational database used to store and manage student records.

## Build & Developer Tools
- **Maven** – Used for dependency management and building the application.
- **Eclipse IDE** – IDEs used for development, debugging, and project management.

## Features
- **Create** – Add new student records.
- **Read** – Display all student details in a structured table.
- **Update** – Modify existing student information.
- **Delete** – Remove student records.
- **Database Integration** – Persistent storage using MySQL.
- **User-Friendly UI** – Clean, responsive design using Bootstrap + Thymeleaf.
